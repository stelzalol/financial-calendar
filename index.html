No iPhone?

You can still use the calendar if your calendar app supports subscribed iCal / ICS calendar feeds.

  • Apple Calendar on Mac, iPhone and iPad
  • Google Calendar and Android
  • Outlook / Microsoft 365
  • Other calendar apps that support subscribed ICS feeds

How to subscribe

Use the live subscription feed where possible. Importing the .ics file directly may create a one-time copy that does not update automatically.

iPhone / iPad

  1. Open Settings.
  2. Go to Calendar → Accounts.
  3. Tap Add Account → Other.
  4. Tap Add Subscribed Calendar.
  5. Paste the feed link and save.

Mac Apple Calendar

  1. Open Apple Calendar.
  2. Choose File → New Calendar Subscription.
  3. Paste the feed link.
  4. Choose your refresh settings and subscribe.

Google Calendar / Android

  1. Open Google Calendar on a computer.
  2. Go to Other calendars → + → From URL.
  3. Paste the feed link.
  4. Add the calendar.
  5. It should then sync to Google Calendar on Android.

Outlook / Microsoft 365

  1. Open Outlook Calendar on the web.
  2. Choose Add calendar.
  3. Select Subscribe from web.
  4. Paste the feed link and save.
  5. Outlook may refresh subscribed calendars more slowly.
Tip: If the subscribe button downloads a file instead of subscribing, copy the feed link and add it manually as a subscribed calendar.

How to unsubscribe

iPhone / iPad

  1. Open the Calendar app.
  2. Tap Calendars.
  3. Tap the info button next to this macro calendar.
  4. Tap Unsubscribe or Delete Calendar.

Mac Apple Calendar

  1. Open Apple Calendar.
  2. Find this calendar in the sidebar.
  3. Right-click it.
  4. Choose Unsubscribe or Delete.

Google Calendar

  1. Open Google Calendar on a computer.
  2. Open Settings.
  3. Select this macro calendar.
  4. Choose Remove calendar → Unsubscribe.

Outlook

  1. Open Outlook Calendar.
  2. Find the subscribed macro calendar.
  3. Right-click it.
  4. Choose Delete Calendar or Remove.
